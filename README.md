**Simple BackEnd App** where Web Scraping is done with Puppeteer to the BCV website. The app has only one route: `'/'`.

**NOTE**: `this is done for practical purposes.`

## 📝 It is used

- Express (v4.18.2)
- Cors (v2.8.5)
- Dotenv (v16.3.1)
- Puppeteer (v20.9.0)

## 🎯 Setup

### 📦 Installation

```shell
pnpm i
```

### 🚀 Run

```shell
pnpm dev
```

### 🔨 Build

```shell
pnpm build
```
